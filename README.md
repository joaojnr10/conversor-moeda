# 💱 Conversor de Moedas

Um **conversor de moedas online** desenvolvido com **HTML, CSS e JavaScript**, que permite converter valores entre diferentes moedas de forma simples e rápida.

Este projeto foi criado para praticar **manipulação do DOM, lógica de programação e formatação de valores monetários** no JavaScript.

---

## 🌐 Acesse o projeto online

🔗 **Demo do projeto:**
https://joaojnr10.github.io/conversor-moeda/

---

## 🚀 Funcionalidades

* Conversão entre diferentes moedas
* Interface simples e intuitiva
* Atualização dinâmica do nome e imagem da moeda
* Formatação automática de valores monetários
* Conversão rápida através de botão

---

## 💰 Moedas disponíveis

* 🇧🇷 Real Brasileiro (BRL)
* 🇺🇸 Dólar Americano (USD)
* 🇪🇺 Euro (EUR)
* 🇬🇧 Libra Esterlina (GBP)
* ₿ Bitcoin (BTC)
* 🇨🇭 Franco Suíço (CHF)

## 🛠️ Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **Intl.NumberFormat** (formatação de moedas)
